# Knowledge Files Storage

Este diretório serve como repositório de recursos para a área de Conhecimentos.
Estruturado baseando-se por tipagens para importações dinâmicas ou renderizações dedicadas na pipeline do Webpack.

- /doc: Arquivos Word, planilhas geradas
- /images: Infográficos e recortes de design
- /pdf: Referências oficiais, RFCs e Whitepapers
- /text: Conteúdos em Markdown e guias textuais puros
